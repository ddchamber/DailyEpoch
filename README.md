The Daily Epoch was a 30-day challenge where I trained myself in the backend architecture of professional-grade AI agents.

Each day focused on learning a new concept in agent development through research on modern AI architecture. After studying the material, I worked with a personalized ChatGPT “expert” trained on that day’s topic to deepen my understanding. The project documents this process through daily themes, AI-generated summaries based on my notes, handwritten research notes, and question-and-answer exercises used to test my understanding.

For the final portion of the challenge, I applied these ideas to design a blueprint for a real-world AI agent called Scout, which is intended to generate in-depth MLB matchup scouting reports. Scout is designed as a structured AI system that retrieves real statistics, analyzes how player or team skills interact, and generates a clear, fan-friendly scouting report supported by evidence and explicit uncertainty.

Overall, this project represents both a learning journal and a system design exercise exploring how modern AI agents can be structured, evaluated, and implemented.
